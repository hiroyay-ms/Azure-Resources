## 仮想ネットワークの作成

### パラメーター
- **Virtual Network Name**: 仮想ネットワーク名

- **Location**: リージョン

- **Address Prefix**: IPv4 名前空間

- **Subnet Name 1**: サブネット名

- **Subnet Prefix 1**: サブネット アドレス範囲

- **Subnet Name 2**: サブネット名

- **Subnet Prefix 2**: サブネット アドレス範囲
